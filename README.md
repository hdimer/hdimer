# Hi, I'm Haïm 👋

I build AI-native tools and ship them. CLIs, agents, MCP servers, and full-stack web apps. Thirty years in, I build and manage the AI agents that do the heavy lifting: directing more than typing, but close enough to the code to know when something's wrong. What I care about hasn't changed: tools that do one thing cleanly, READMEs a stranger can follow, and code another engineer understands in sixty seconds.

Below is what I've been making lately. If you'd rather ask than read, my site answers questions about me in my own words: **[haimdimer.com](https://haimdimer.com)**.

### What I'm building

- **[earshot](https://github.com/hdimer/earshot)** — Turns books into multi-episode podcasts. Next.js 16, Neon, and Cloudflare R2, built to actually listen to.
- **[code-index](https://github.com/hdimer/code-index)** — Semantic code search as an MCP server, backed by Milvus. So an agent can find code by meaning, not just grep.
- **[live-poll-kit](https://github.com/hdimer/live-poll-kit)** — Drop-in live polling with QR codes for talks and classrooms. Vercel plus Neon, five-minute setup.

### How I work

Experience is what lets me steer agents well: knowing what good looks like, catching the wrong turn three commits early, and insisting on simple over clever, small reversible changes, and tests before code. Thirty years of building didn't make me precious about typing every line; it made me better at knowing which lines matter. The repos here are the output of that partnership.

### Beyond the terminal

When I'm not shipping, I'm usually documenting. I've photographed Mediterranean sea rescues, the war in Ukraine, and Israel and Gaza. And one of my favorite projects started as a prediction game I built with my kids.
